Conhecimentos gerais de automação:

1 – Descreva o que é o padrão Page Object?
R:
O padrão Page Objects tem um conceito bem simples, a proposta é de criar um objeto para cada página/tela, e utilizando
a boa e velha orientação a objetos para encapsular em cada classe os atributos e métodos, como campos e ações de cada página.

2 – O que são mocks, e quando devem ser utilizados?
R:
Mocks são imitações ou unidades falsas que simulam o comportamento de unidades reais.
Usamos o mock quando queremos saber se uma função vai ser chamada corretamente, 
quantas vezes ela vai ser chamada, se os parâmetros esperados são os corretos.


3 – Cite 3 tipos de testes que podem ser automatizados em API e 3 para Front end.
R:
Testes API
- Teste de gravação de dados em banco de dados
- Teste de consulta de dados 
- Teste de exclusão de dados

Teste Front end
- Teste de preencher formularios
- Teste de login efetuado com sucesso
- Teste de preencher campos
   - Exemplos:
Validar tamanho maximo de inserção de caracteres
Validar se aceita letras e numeros
Validar se aceitar caracteres especiais
